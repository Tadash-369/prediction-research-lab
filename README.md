# Prediction Research Lab / 分析研究所

ロト6・ロト7を中心に、予測手法の研究・検証・改善を継続するための分析アプリです。

目的は当選や利益を保証することではありません。  
目的は、複雑またはランダムな現象に対して、どの予測手法が有効かを継続的に研究・評価・改善することです。

## 対象

### Phase 1
- ロト6
- ロト7

### Phase 2
- 日経平均
- TOPIX
- S&P500
- NASDAQ
- ドル円

### Phase 3
- 配当株
- 個別株

## 基本方針

この研究所では、単に予想を出すだけではなく、以下のサイクルを重視します。

1. 予想
2. 結果
3. 検証
4. 改善

すべての予想履歴、実結果、検証結果、改善案を研究資産として保存します。

16. ランダム予測モデル

## 主なファイル

- `analysis_research_lab.py`  
  研究所トップ画面。ロト6、ロト7、AI改善、投資、保守を横断表示します。

- `arl_research_engine.py`  
  分析モデル、スコア計算、検証、貢献度分析、AI改善サマリーの中核処理です。

- `prl_maintenance.py`  
  CSV補完、バックアップ、モデル名正規化、フォルダ同期などの保守処理です。

- `loto6_streamlit_app.py`  
  ロト6専用の分析ダッシュボードです。

- `loto7_streamlit_app.py`  
  ロト7専用の分析ダッシュボードです。

- `loto6.csv`  
  ロト6の当選履歴データです。

- `loto7.csv`  
  ロト7の当選履歴データです。

## セットアップ

Python環境で以下を実行します。

```bash
pip install -r requirements.txt
```

## 起動方法

研究所トップ画面を起動する場合:

```bash
streamlit run analysis_research_lab.py
```

Windowsでは、以下のバッチファイルから起動できます。

```bat
start_prediction_research_lab.bat
```
